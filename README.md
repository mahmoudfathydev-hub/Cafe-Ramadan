## ğŸ“¦ Installation & Setup

### Prerequisites

- Node.js 18+ installed
- npm, yarn, pnpm, or bun package manager

### Installation Steps

1. **Clone the repository**

```bash
git clone https://github.com/mahmoudfathydev-hub/Cafe-Ramadan.git
cd Cafe-Ramadan
```

2. **Install dependencies**

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

3. **Run development server**

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Manual Build

```bash
npm run build
npm start
```
